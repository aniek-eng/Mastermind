# Mastermind

Here you can find our improved figure and our codes. The file with the programmes has the name "codes (1)."

These are the codes from our project. 

To play the game, with the computer as codemaker, run: play_mastermind()

To use Knuth's algorithm to break the code, run: knuth()

To use our intelligent algorithm to break the the code, run: intelligent_mastermind_guide(n,k,repeating, zeros, printing), where:
- n is the length of the code
- k is the number of colours/digits allowed in the code
- repeating is set to "off" or "on" depending on whether repetition of colours/digits is allowed in the code
- zeros is set to "off" or "on" depending on whether zeros are allowed in the code
- printing is set to "off" or "on" depending on whether you want the programme to print "Examining element... of..." to give an indication of the the progress
